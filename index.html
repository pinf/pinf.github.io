PINF Platform for JavaScript - github.com/pinf

The PINF Platform rooted in JavaScript is the result of many years of work by Christoph Dorn with a heavy focus on flexible toolchain design so you can change your mind at any time. This has been a long-time solo effort which I hope to gradually grow without compromising my specific vision for it.

At the core of the PINF Platform for JavaScript is the:

PINF JavaScript Loader - Optimized JavaScript Bundle Loader

  • Source: github.com/pinf/pinf-loader-js
  • Demo: pinf.github.io/pinf-loader-js/workspace/www/index.html

The loader is at the core of the platform and represents the minimal footprint for an instance. It provides a nestable URI-based namespacing system used to load code bundles that implement all features of an application or system using networked modules, components and services.

Putting a namespacing system that overlaps with the top-level-domain space at the core of the platform and thus each application instance opens up a world of possibilities. It elevates application and system development to component orchestration around pure data-based (vs code) system models that drive system manifestation.
